Allow Active Content

If the e‑learning course is located on a CD or on your computer (not on a Web site), you must take into consideration that Windows Internet Explorer 7 and latest versions has restricted files from running active content from unknown sources that could harm your computer.

When inserting the CD your confirmation is required to allow active content from the CD to run on your computer. If you answered Yes you can start this e‑learning course. If you answered No then follow these steps to allow active content:

  1. On the Tools menu, click Internet Options.
  2. In the Internet Options dialog box, click the Advanced tab.
  3. Scroll the Settings list until the Security category is displayed.
  4. Check whether these options are ticked:
  5. Allow active content from CDs to run on My Computer — if the e‑learning course is located on a CD;
  6. If the required option isn't ticked select the appropriate check box.
  7. Restart Internet Explorer in order to changes take effect.
